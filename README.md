# Precision Invest

## Sobre o projeto

O Precision Invest é uma plataforma web desenvolvida para facilitar a tomada de decisões financeiras. O sistema permite simular investimentos, acompanhar uma carteira de ações e registrar operações de forma organizada.

O projeto foi desenvolvido de forma independente, com foco em aprendizado prático e aplicação em cenários reais.

## Funcionalidades

* Sistema de autenticação de usuários
* Simulação de investimentos em renda fixa
* Gerenciamento de carteira de ações
* Integração com dados reais do mercado financeiro (Yahoo Finance)
* Registro e histórico de operações
* Cálculo de indicadores financeiros

## Tecnologias utilizadas

* Python (Flask)
* SQLite
* HTML e CSS (Tailwind)
* Biblioteca yfinance

## Objetivo

O objetivo do projeto é auxiliar usuários, especialmente iniciantes, a compreender melhor o funcionamento dos investimentos e apoiar a tomada de decisões financeiras.

## Como executar

1. Execute o arquivo app.py
2. Acesse http://localhost:5000 no navegador
